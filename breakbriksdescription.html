Tic Tac Toe

Features

  1. Responsive Design:

    • Mobile-Friendly: Players can tap or swipe to control the ball.
    • Desktop-Friendly: Players can use arrow keys for movement and the Enter or Up arrow key to blast bricks.

  2. Game Mechanics:

    • A ball moves left and right automatically, and players can blast bricks by creating additional balls.A ball moves left and right automatically, and players can blast bricks by creating additional balls.
    • Falling bricks are randomly generated, and players must break them to score points.
    • The game keeps track of the highest score using local storage.

  3. User Interface:

    • The game includes a start button, score display, and highest score display.
    • Clear instructions are provided for both mobile and desktop users.

  4. Interactive Elements:

    • Ball and brick elements are visually distinct and easy to identify.
    • Controls are intuitive, with touch/mouse and keyboard inputs seamlessly integrated.
  • Mobile:
    • Move Left: Tap or swipe the left side of the screen.
    • Move Right: Tap or swipe the right side of the screen.
    • Blast Bricks: Tap the middle of the screen.
  • Mobile:
    • Move Left: Use the left arrow key.
    • Move Right: Use the right arrow key.
    • Blast Bricks: Use the up arrow key or press Enter.

Technical Highlights:

  • HTML & CSS:Structured layout and responsive design to ensure a smooth user experience across devices.
  • JavaScript:Implements game logic, including ball movement, brick creation, and collision detection.
  • Local Storage:Saves and displays the highest score achieved by the player.

Challenges and Learning:

  • Responsive Design: Ensuring the game is playable on both mobile devices and desktops required careful planning and implementation.
  • Game Logic: Developing the logic for ball movement and collision detection was a rewarding challenge.
  • User Experience: Creating an intuitive and engaging user interface was a key focus.

Conclusion:

Developing "Break the Bricks" was a fantastic experience that helped me improve my coding skills and learn more about game development. I hope you enjoy playing the game as much as I enjoyed creating it.
