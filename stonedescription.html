Stone Paper Scissor

Project Overview

I recently developed a web-based Rock Paper Scissors game. This project allowed me to enhance my skills in HTML, CSS, and JavaScript, and create an engaging and interactive user experience. Below are the key features and functionalities of the game.

Features

  1. Interactive UI:

    • The game has a visually appealing user interface with background images and buttons that change appearance when hovered over.
    • Users can easily select their moves (stone, paper, or scissors) with corresponding buttons.

  2. Level Selection:

    • Players can choose the number of rounds they want to play from a dropdown menu.
    • Options range from 5 to 50 rounds, providing flexibility for different game lengths.

  3. Gameplay:

    • The game displays the user's and computer's choices with images.
    • Results of each round (win, lose, draw) are shown immediately.
    • The score is updated dynamically after each round.

  4. Scoreboard:

    • A scoreboard tracks the scores of both the user and the computer.
    • The game announces the winner and updates the background image accordingly when the chosen level is reached.

  5. Restart Functionality:

    • Users can restart the game at any time, resetting the scores and level selection.

How It Works

  • User Interface: The game interface is centered within a styled div box, providing a clear and structured layout.
  • Level Selection: Upon selecting a level, the game begins counting rounds.
  • Player Moves: Players click on the stone, paper, or scissors buttons to make their move. The computer's move is randomly generated.
  • Result Calculation: The game logic determines the winner of each round and updates the results and scores.
  • End of Game: When the user's or computer's score reaches the selected level, the game announces the winner and updates the background image to indicate the result.

Technologies Used

  • HTML: For structuring the game layout.
  • CSS: For styling the game elements, including buttons, images, and the overall layout.
  • JavaScript: For handling game logic, user interactions, and dynamic updates.

Conclusion

This Rock Paper Scissors game project was a fun and educational experience. It helped me improve my coding skills and understand how to create a complete interactive application. I hope you enjoy playing it as much as I enjoyed making it!
