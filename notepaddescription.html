NotePad

Main Components:

  • A Frame: Contains a TextArea for text input.
  • MenuBar: Includes menus for File, Edit, Format, and Color.

Menu Functionalities:

  • File Menu:
    • New: Creates a new text file.
    • Print: (Functionality pending in the provided code.)
    • Exit: Closes the application.
  • Edit Menu:
    • Copy, Paste, Cut: Standard clipboard operations.
    • Undo, Redo: Basic undo and redo operations.
    • Find: Searches for text within the document.
    • Replace: Replaces text in the document.
    • Delete: Deletes selected or all text.
  • Format Menu:
    • Font selection and size adjustment (UI prepared for expansion).
  • Color Menu:
    • Background and text color customization.

Key Features:

  • Real-time updates for word count, line count, and character count.
  • Keyboard shortcuts (e.g., Ctrl + N, Ctrl + W) for ease of use.
  • Find and Replace functionality with dialog windows for input and actions.
  • Customizable font and color settings.

Event Handling:

  • Listens for key events (e.g., shortcuts like Ctrl+F for Find).
  • Handles menu item actions using ActionListener.

UI Design:

  • Dialogs for Find and Replace functionalities with colorful backgrounds.
  • Dynamically updating status bar for text statistics.
