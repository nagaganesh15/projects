Tic Tac Toe

Overview

The TicTacToe Game is a classic two-player game implemented as a web-based application. It allows players to enjoy a simple yet engaging game of TicTacToe, complete with player names, a reset button, and visual feedback for wins or draws.

Features

  • Interactive User Interface: The game has a clean and interactive design with large, clickable buttons for easy play.
  • Player Names: Players can enter their names, which are displayed when a player wins.
  • Reset Functionality: The game can be reset at any time, clearing the board and player names.
  • Winning and Draw Detection: The game automatically detects and announces the winner or a draw.
  • Background Images: The game features a changing background image upon winning to celebrate the victory.

Technologies Used

  • HTML: Provides the structure of the game interface.
  • CSS: Styles the game elements, including buttons, background, and layout.
  • JavaScript: Implements the game logic, including player moves, win/draw detection, and reset functionality.

How to Play

  1. Enter Player Names: Enter the names of Player 1 and Player 2 in the provided input fields
  2. Start Playing: Players take turns clicking on the buttons to mark their moves. Player 1 uses 'X' and Player 2 uses 'O'.
  3. Win Detection: The game will automatically detect a win or draw and display the result.
  4. Reset the Game: Click the "Reset" button to start a new game.

Conclusion

The TicTacToe Game project is a great example of using HTML, CSS, and JavaScript to create an interactive and enjoyable web application. It provides a fun way to demonstrate basic game logic and user interaction in a browser environment. This project is suitable for showcasing in a portfolio to highlight skills in web development and game design.
