Alarm Application

This mini-project demonstrates the creation of a simple alarm application using Java's AWT (Abstract Window Toolkit) and sound libraries. The project focuses on real-time functionality, user interaction, and basic multimedia integration.

Key Features:

  • Set Alarm: Users can input the desired time in the HH:mm format to set an alarm.
  • Stop Alarm: The application provides a button to stop the alarm once it starts ringing.
  • Real-Time Monitoring: Continuously checks the current time against the user-set alarm time.
  • Audio Integration: Plays an alarm sound ("alarm.wav") when the set time matches the system time.

User-Friendly Interface:

  • Color-coded Elements: Clear distinction between input, action, and display areas with a colorful design.
  • Interactive Controls: Includes buttons for setting and stopping the alarm, ensuring ease of use.

Technical Details:

  • GUI Design: Built using AWT components like Frame, Label, TextField, and Button.
  • Real-Time Clock: Utilizes LocalTime and DateTimeFormatter from the java.time package to handle time operations.
  • Audio Playback: Integrates the AudioSystem and Clip classes to play the alarm sound.
  • Event Handling: Implements ActionListener for button clicks and WindowAdapter for managing the window close event.

Learning Outcomes:

  • Enhanced understanding of GUI development with AWT in Java.
  • Experience in real-time data handling using Java's java.time package.
  • Basic multimedia programming with sound playback in Java.
  • Event-driven programming through ActionListener and WindowAdapter.

This project showcases my ability to design simple yet functional applications and serves as a foundation for exploring more advanced Java-based GUI and multimedia projects.
