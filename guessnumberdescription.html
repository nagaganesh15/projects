Guess the Number

Features:

  1. Responsive Design:

    • The game is designed to be user-friendly across various devices, ensuring a smooth experience on both mobile phones and desktops.

  2. Game Mechanics:

    • Players can select different levels of difficulty, choosing a range between 1 to 50, 1 to 100, up to 1 to 500.
    • The game generates a random number within the chosen range, and players have to guess the number.
    • If the guess is incorrect, hints are provided to help narrow down the possibilities.
    • The score is tracked and displayed, with the highest score saved using local storage.

  3. User Interface:

    • Clear and vibrant interface with interactive elements such as buttons and number input.
    • Instructions and hints are displayed dynamically to guide the player.

  4. Interactive Elements:

    • Players can interact with the game through number input and buttons for checking their guess and submitting the correct answer.
    • Feedback is given in the form of text hints and audio cues for correct and incorrect guesses.

How to Play:

  1. Select Level: Choose your guessing level from the dropdown menu.
  2. Guess the Number: Enter your guess in the number input field and click "Check" to see if it's correct.
  3. Hints: If your guess is incorrect, hints will be provided:
    • The sum of the digits of the correct number.
    • The number of digits in the correct number.
  4. Submit: If your guess is correct, click "Submit" to confirm and start a new round.
  5. Score Tracking: Your current score and highest score are displayed at the top.

Technical Highlights:

  • HTML & CSS: Provides a structured layout and responsive design for a seamless user experience.
  • JavaScript: Implements the game logic, including number generation, user input handling, and hint generation.
  • Local Storage: Utilized to save and display the highest score achieved by the player.

Challenges and Learning:

  • Responsive Design: Ensuring the game is visually appealing and functional on both mobile and desktop devices.
  • Game Logic: Developing the logic for generating random numbers and providing meaningful hints.
  • User Experience: Creating an engaging and intuitive interface with immediate feedback for the player.

Conclusion:

Developing "Guess the Number" was a fantastic learning experience that helped enhance my coding skills and understanding of game development. I hope you enjoy playing the game and find it as challenging and fun as I intended.
